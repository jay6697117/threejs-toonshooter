# 三国卡通射击对战游戏 - 完整设计文档

> 基于 Tiny Toon Duel 架构的三国风格重制版

---

## 一、设计概述

### 1.1 核心定位

| 要素 | 设计决策 |
|------|----------|
| **风格** | 三国卡通Q版射击对战 |
| **战斗节奏** | 混合风格（不同武器不同节奏） |
| **武器分类** | 按射程分类（近/中/远/特殊） |
| **武器机制** | 元素效果 + 蓄力机制 + 弹道特性 |
| **武器规模** | 丰富版（15种主武器） |
| **副武器** | 混合系统（火药/机关/道具共9种） |
| **角色差异** | 纯外观差异（12武将，无属性差别） |
| **场景数量** | 10个场景轮换 |
| **开发范围** | 完整版直接开发 |

## 二、武器系统

### 2.1 近程武器（4种）

射程短、出手快、适合近身缠斗

| 武器 | 伤害 | 射速 | 射程 | 弹匣/备弹 | 换弹 | 弹道特性 | 特殊机制 | 状态效果 |
|------|------|------|------|-----------|------|----------|----------|----------|
| **飞刀** | 18 | 2.5/秒 | 12m | 6/24 | 0.8秒 | 直线瞬发 | 连续投掷 | 无 |
| **飞镖** | 12×3 | 1.5/秒 | 14m | 8/32 | 1.0秒 | 直线快速 | 蓄力扇形散射 | 减速1.5秒 |
| **袖箭** | 14×2 | 3.0/秒 | 10m | 4/16 | 0.6秒 | 直线瞬发 | 双发连射 | 中毒3秒 |
| **回旋刃** | 22 | 1.2/秒 | 15m | 1/8 | 0.5秒 | 飞出后返回 | 可命中2次 | 撕裂2秒 |

**设计理念**：
- 飞刀：稳定可靠的基础武器
- 飞镖：蓄力换取范围压制
- 袖箭：刺客风格高爆发 + 毒伤
- 回旋刃：高技巧上限
